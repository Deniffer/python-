# Jupyter Notebook 使用小知识
## 命令模式
>快捷键 shirt+enter 执行当前块并生成下一个空白的代码块，进入命令模式
>快捷键 control+enter 仅执行当前代码块
>快捷键  Alt+enter  执行当前代码块并生成下一个空白块，并进入下一个空白代码块的编辑模式
>快捷键 D+D         处于命令模式 D+D 删除当前代码块
		L          显示行号
		ESC        激发命令模式
		Z          回复最后一个删除的代码块	
		shirt+M    合并选中代码块

## 编辑模式
>快捷键 shirt+Tab 提示更多信息，+?后显示更加详细的信息
